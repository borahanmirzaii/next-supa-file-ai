# User Guide

## Getting Started

### Sign Up

1. Navigate to the login page
2. Enter your email address
3. Enter a secure password (minimum 8 characters)
4. Click "Sign Up"
5. Check your email for verification link
6. Click verification link to activate account

### Login

1. Enter your email and password
2. Click "Login"
3. You'll be redirected to the dashboard

## File Upload

### Upload Files

1. Navigate to the Files page
2. Drag and drop files onto the upload area, or click to browse
3. Supported file types:
   - PDF documents
   - Word documents (DOCX)
   - Text files (TXT)
   - Images (PNG, JPG, WebP)
   - Excel files (XLSX, CSV)
   - Code files (JS, TS, etc.)
4. Maximum file size: 50MB
5. Files will upload automatically
6. Progress is shown in real-time

### View Files

- All uploaded files appear in the file list
- Files show name, size, type, and upload date
- Click "View" to see file details
- Click "Download" to download file
- Click "Delete" to remove file

## AI Analysis

### Analyze Files

1. Upload a file
2. Click "View Analysis" on the file
3. Analysis starts automatically
4. Progress updates in real-time
5. Results include:
   - Summary
   - Key points
   - Insights
   - Metadata

### Analysis Types

- **Document Analysis**: Extracts text, summarizes content, identifies key topics
- **Image Analysis**: Describes images, identifies objects, extracts text
- **Code Analysis**: Analyzes code structure, identifies patterns, suggests improvements
- **Data Analysis**: Summarizes data structure, identifies trends, extracts key findings

## Knowledge Base

### Chat with Your Files

1. Navigate to Knowledge Base
2. Switch to "AI Chat" tab
3. Ask questions about your uploaded files
4. AI responds using information from your files
5. Citations show which files were used

### Semantic Search

1. Switch to "Search" tab
2. Enter search query
3. Results ranked by relevance
4. Similarity scores shown
5. Filter by specific files

### File Filtering

- Use file filter dropdown
- Select specific file to search within
- Select "All Files" to search all

## Integrations

### Connect Platforms

1. Navigate to Integrations page
2. Click "Connect" on desired platform
3. Complete OAuth flow
4. Platform status changes to "Connected"
5. Tools become available

### Execute Tools

1. Browse available tools
2. Select a tool
3. Click "Execute"
4. Fill in required fields
5. Click "Run"
6. View results

### Disconnect Platform

1. Click "Disconnect" on platform
2. Confirm disconnection
3. Platform status changes to "Disconnected"
4. Tools no longer available

## FAQ

### How do I delete my account?

Contact support to delete your account and all associated data.

### What file types are supported?

PDF, DOCX, TXT, images (PNG/JPG/WebP), Excel (XLSX/CSV), and code files.

### What's the file size limit?

50MB per file.

### How long are files stored?

Files are stored indefinitely until you delete them.

### Is my data secure?

Yes, all files are encrypted and stored securely. Only you can access your files.

### Can I share files?

Currently, files are private to your account. Sharing features coming soon.
